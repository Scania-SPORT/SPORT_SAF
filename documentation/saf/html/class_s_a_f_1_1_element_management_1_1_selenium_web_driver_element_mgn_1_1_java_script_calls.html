Execute the java script, scriptToExecute, on the site driver, e.g. to simulate an action that cannot be achieved by a webdriver action. Such actions could be as forcing the browser to show hidden elements, get browser information, etc. The java script execution should be used with caution since it not an exact simulation of the user behavior.
